# 주제
멀티에이전트 기반 학습 지원 시스템

# 개요
멀티에이전트 기술을 활용하여 사용자의 능동적 학습을 지원하고 학습 효율을 높일 수 있는 시스템 개발

# 요구사항
1. 단순히 교정이 아니라 구체적인 평가를 수행하고 관련 문제 제시를 통해 학습을 통한 실질적 개선을 유도하는 시스템 개발
2. 학습 이론에 기초한 피드백 제시 및 문제 제시가 이뤄져야 하며, 적용 주제는 작문, 논술, 언어학습 등 자유롭게 고려 가능
3. 학습 피드백은 시각적으로 제시되도록 할 것(프론트엔드)
4. 단순 상용 API 활용은 지양할 것. 소형 언어모델이라도 자체 서버를 두고 API 활용할 것(백엔드)
5. 기존 제품/서비스 대비 명확한 차별점을 제시할 것
6. 사용 전/후에 대한 통계 분석이 이뤄져야 함
