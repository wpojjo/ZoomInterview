# 발표 자료 제작 기준

## 발표 맥락

- **행사**: 대학교 창의설계경진대회
- **형식**: 발표 영상 제출 (최대 7분)
- **청중**: 교수님, 대학생

## 영상 구성

7분을 두 파트로 분리:

| 파트 | 길이 | 내용 |
|------|------|------|
| 후킹 영상 | ~1분 | 문제 공감 + 서비스 전체 플로우 데모 |
| PPT 발표 | ~6분 | 슬라이드 상세 설명 |

후킹 영상과 PPT의 역할 분담:
- 후킹 영상: 문제 공감(면접 후 막막함) → 서비스 전체 흐름 데모 (스토리 중심)
- PPT: 기능별 조각 클립/스크린샷 (상세 중심)

## 디자인 기준

- **메시지**: 슬라이드 하나에 핵심 메시지 하나
- **텍스트**: 꽉 채우지 않고 핵심 한 줄 위주로 크게
- **이미지**: 비중 높게, 배경 사진 적극 활용
- **컬러**: 2~3가지로 제한 (딥 네이비 / 화이트 / 포인트 블루)
- **폰트**: 2개 이하 (제목 Bold 산세리프, 본문 Light)
- **여백**: 빈 공간을 두려워하지 않음
